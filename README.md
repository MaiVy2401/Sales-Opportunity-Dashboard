## Sales Opportunity Dashboard — Power BI

Interactive two-page dashboard tracking 1,819 sales opportunities worth $5.37M.

### Features
- Pipeline stage analysis across Validated, Identified, Proposal, Qualified, Won, Lost
- Sales channel breakdown (Telesales, F2F, Partners, Website)
- Geographic opportunity map across 10+ countries
- Salesperson drill-through page with individual KPIs and transaction detail
- Dynamic slicers by Year, Month, and Country

### Tools Used
- Power BI Desktop
- DAX measures
- Cross-page slicers and drill-through
